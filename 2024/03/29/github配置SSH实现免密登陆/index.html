Git配置SSH免密登陆

跟着官网文档操作的：https://docs.github.com/zh/authentication/connecting-to-github-with-ssh/about-ssh

1.第一步-设置全局用户名和邮箱

用户名：git config –global user.name “neme”
邮箱：git config –global user.email “email”
设置后查看是否设置成功：
git config –global user.name
git config –global user.email

2.第二步-生成密钥

首先进入用户主目录：cd ~
配置免密登陆命令：ssh-keygen -t rsa -C “email”
按三次回车，会显示密钥自动保存到默认位置
按照官方文档中的指引在ssh-agent中添加生成的新的ssh密钥（详细步骤看官方文档，这一步不是必须的）
进入公钥文件：cd ~/.ssh/id_rsa.pub
复制文件里的所有内容（即公钥）
进入github—>setting—>ssh and gpg keys，选择new ssh keys，将公钥复制上去
测试ssh连接：ssh -T git@github.com
输入yes，如果返回信息中出现了正确的用户名，说明连接没有问题
